# CCC — Content/Copy Management System

CCC is a multi-tenant CMS for managing translation keys and their values across multiple languages. It enables developers, project managers, and editors to define labels, dynamic texts, mail templates, and other static content consumed by client applications.

---

## Quick Start

### Backend

```bash
dotnet build CCC.sln
dotnet run --project CCC/CCC.csproj
```

Swagger UI is available at `http://localhost:5039/swagger` in development.
